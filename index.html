Web Font Example

This page uses inlined web fonts in a CSS file that was generated via webfont-dl.

The CSS contains three fonts and is approximately 90kB in size, 70kB when gzipped.

p> 
webfont-dl -d "http://fonts.googleapis.com/css?family=Crimson+Text:400,400italic|Raleway:500" -o font/font.css \
	--font-out font/fonts --css-rel=fonts
